### 使用说明：
前置要求：
  - 需要上级文件目录包含 lib 文件夹 (lib 文件夹中包含对应的动态链接库)  
  - 本程序目前为 Windows 使用

        1.修改对应 COM 口
        2.cd scripts
        3.python 需要运行的程序.py 

文档说明：
- mode.py 用于测试各种模式  

      cd scripts
      python mode.py
      
      motor_s1_start = 速度模式
      motor_p1_start = 位置模式
      motor_t1_start = 力矩模式

      空格 = 停止测试  
      右键 = 增加  
      左键 = 减少  
    
- changeID.py 用于修改电机ID  

      cd scripts
      python changeID.py

      运行程序后电机进入 电子棘轮模式，用力转动点击会有顿挫感
      转动卡顿1次为 ID 0
      转动卡顿2次为 ID 1
      转动卡顿3次为 ID 2

      按下空格完成 ID 设置



宇树A1电机说明：  
 - 电机最多支持3个串联
 - ID=0,1,2,0xBB (对所有电机广播)
 - Mode
   - 0=停转
   - 5=开环
   - 10=闭环
   - 11=修改ID
 - 电机接收到控制指令才会返还数据
   
  
宇树A1电机参数:
  |参数|数值|
  |--- |---|
  |最大扭矩|33.5 Nm|
  |减速比|9.1:1|
  |波特率|4.8MBd = 4800000 Baud Rate|
  |串口数据位|8bit|
  |串口停止位|1bit|
  |校验|无奇偶校验|
