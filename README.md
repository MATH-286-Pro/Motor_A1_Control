### 使用说明：
前置要求：需要上级文件目录包含 lib 文件夹 (lib 文件夹中包含对应的动态链接库)  

    1.修改对应 COM 口
    2.cd scripts
    3.python check.py 

使用说明：
- test.py 用于测试速度模式
- pos.py  用于测试位置模式

宇树A1电机说明：  
 - 电机最多支持3个串联
 - ID=0,1,2,0xBB
 - Mode
   - 0=停转
   - 5=开环
   - 10=闭环
 - 电机接收到控制指令才会返还数据
   
  
宇树A1电机参数:
  |参数|数值|
  |--- |---|
  |最大扭矩|33.5 Nm|
  |减速比|9.1:1|
  |波特率|4.8MBd|
  |串口数据位|8bit|
  |串口停止位|1bit|
  |校验|无奇偶校验|
